Class Ligte

Ever wanted to play a 2D game with infinite world generation? If yes, this is the game for you!

|| Introduction

Class Ligte is a roguelite game where you collect items and slay monsters and bosses, but the core mechanic of Class Ligte is the exploration.
The exploration is done through a series of worlds. each worlds provides you with new items and one more skill.
These skills can be all the way from fishing to taming.

Worlds

In ClassLigte you are greeted with the world system.
Every world takes a total of 2000 tiles except for the last world which goes on forever.
Every world also introduces a new skill. Most of the time you will be able to craft a tool for that skill in the previous world. For example: You can craft a pickaxe at world 2, only at world 3 this pickaxe will be used.

|| How to change your world version

Before changing your world version you should always create a backup. Upgrading a world to a newer version can corrupt data and make your world unplayable.

Now how to change your world version:
When a world is opened in the wrong version it will give a message like the one below:
World is not in version [Your current world version], while it must be in version '[Required version]'.
To change your world version you need to go to your appdata folder (or .config folder on Linux) and locate the "ClassLigte-Sverryboiii" folder. In there lies all the game data.
In there you go to the "worlds" folder and open the world folder you want to upgrade. After that locate the JSON file named "config_data" and open it in notepad (or whatever text editor you use)
Now that you're in the JSON file you need to locate the "version" tag and change the value of that tag to the required version.
(Downgrading works the same, but is considdered more dangerous!)

After that you have upgraded your game!

Things that will and wont change when upgrading / downgrading a world:
If the version you changed your world to has more worlds than your old world they will be automatically added. Only the tiles changed by you or structures (anything that isn't terrain, so trees too) will stay, because these are considdered "important" to the player. When downgrading biomes that dont exist in that version will be converted to biomes that do exist.
When upgrading you will keep all your items, but when downgrading your items may get lost.
But the advice is to never change a world's version unless you really know what you're doing.
